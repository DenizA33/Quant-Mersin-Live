# 📈 Quant-Mersin: Interactive Financial Analysis Dashboard

This project is a **Real-Time Financial Data Analytics** tool built with **Python** and **Streamlit**. It aims to provide actionable insights for stock market participants by filtering noise and identifying trends.

## 🚀 Key Features
- **Live Data Acquisition:** Fetches real-time market data via `yfinance`.
- **Algorithmic Signal Processing:** Implemented **EMA (20/50)** crossover logic to detect bullish and bearish trends.
- **Interactive UI:** A user-friendly dashboard allowing users to analyze any stock ticker (e.g., SISE.IS, THYAO.IS).
- **Automated Insights:** Instant visual alerts for "Buy" and "Sell" zones.

## 🛠️ Tech Stack
- **Languages:** Python (Pandas, NumPy, Matplotlib)
- **Frontend/Dashboard:** Streamlit
- **API:** Yahoo Finance API

## 💼 Why This Project?
Designed as a bridge between **Quantitative Finance** and **Modern Web Applications**, this tool demonstrates my ability to process complex data sets and deliver them through a professional interface.
